# Tejespofájúbot
Tejespofájúcica szerverén a Discord bot source codeja. Nem bonyi


## Run Locally

Clone the project

```bash
  git clone https://github.com/chlkrisz/tejes-bot
```

Go to the project directory

```bash
  cd tejes-bot
```

Install dependencies

```bash
  npm install
```

Start the main file

```bash
  npx ts-node src/index.ts
```
